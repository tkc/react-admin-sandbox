# React Admin

## Local Serve 

```
yarn staet
```

## Requirements

### Node.js

```sh
nvm install v10.x.x
nvm use v10.x.x
```

### Yarn

```sh
brew install yarn
```

## Development

```sh
yarn start
```

## Test

```sh
yarn test
```

### TypeScript compilation check

```sh
yarn tsc
```

### Lint

```sh
yarn lint
```

### Build Development

```sh
yarn build
```

### Build production

```sh
build:prod
```
