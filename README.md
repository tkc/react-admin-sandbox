# React Admin Sandbox

## Requirements

### Node.js

```sh
nvm install v10.x.x
nvm use v10.x.x
```

## Development

```sh
npm run start
```

## Test

```sh
npm run test
```

### TypeScript compilation check

```sh
npm run tsc
```

### Lint

```sh
npm run lint
```

### Build Development

```sh
npm run build
```

### Build production

```sh
npm run build:prod
```
